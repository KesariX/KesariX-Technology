⚡ Global Animations Reference
text

PAGE LOAD SEQUENCE (GSAP Timeline):
  1. Loader screen: Logo draws/morphs in → fades out (0.8s)
  2. Nav items stagger drop in
  3. Hero headline SplitText character reveal
  4. Hero image scale up from 0.9 → 1
  5. CTA buttons fade + slide up
  6. Scroll unlock

SCROLL ANIMATIONS (GSAP ScrollTrigger):
  - All section headings: SplitText line-by-line reveal
  - Cards: stagger fade + translateY from 40px
  - SVG path drawing (Process section)
  - Parallax on hero background elements
  - Horizontal pinned scroll (Projects section)
  - Counter animations (Stats section)

CURSOR (Custom):
  - Default: small white dot + larger ring follower
  - On CTA buttons: expands + shows label
  - On project cards: transforms to "VIEW →"
  - On drag section: transforms to "DRAG"

HOVER MICRO-INTERACTIONS (Framer Motion):
  - Buttons: spring physics, scale 0.97 on press
  - Cards: translateY(-8px), shadow deepen
  - Nav links: magnetic pull effect
  - Images: subtle scale(1.03) zoom

ANIME.JS USES:
  - Hero badge blinking dot
  - Floating stat cards
  - Particle background (CTA section)
  - Number countUp animations
  - SVG icon line draw on section entry
  - Marquee acceleration/deceleration
